# String Processor

Простое веб-приложение для обработки строк различными методами. Введите слово, выберите нужные методы обработки и получите результаты.

## Что это такое?

Одностраничное веб-приложение, которое позволяет:
- Ввести любую строку (слово, ссылку и т.д.)
- Выбрать один или несколько методов обработки
- Получить результаты обработки для каждого выбранного метода

## Доступные методы обработки

- **Реверс строки** - переворачивает строку задом наперед
- **Верхний регистр** - преобразует все буквы в заглавные
- **Нижний регистр** - преобразует все буквы в строчные
- **Удаление пробелов** - убирает все пробелы из строки
- **Заглавная первая буква** - делает первую букву заглавной

Методы загружаются автоматически из backend, так что можно легко добавить новые.

## Технологии

- **Backend**: FastAPI (Python)
- **Frontend**: HTML, CSS, JavaScript (без фреймворков)
- **Сервер**: Uvicorn

## Установка и запуск

### 1. Клонируйте репозиторий (если нужно)

```bash
git clone <ваш-репозиторий>
cd InformationSecurityProject
```

### 2. Создайте виртуальное окружение

```bash
python3 -m venv venv
```

### 3. Активируйте виртуальное окружение

**На macOS/Linux:**
```bash
source venv/bin/activate
```

**На Windows:**
```bash
venv\Scripts\activate
```

### 4. Установите зависимости

```bash
pip install -r requirements.txt
```

### 5. Запустите сервер

Из корня проекта:
```bash
uvicorn src.main:app --reload
```

Или из папки `src`:
```bash
cd src
python main.py
```

### 6. Откройте в браузере

Перейдите по адресу: `http://localhost:8000`

## Как использовать

1. Введите слово или строку в поле ввода
2. Выберите один или несколько методов обработки (кнопки станут синими при выборе)
3. Нажмите "Отправить"
4. Результаты появятся в окне снизу - для каждого метода отдельно
5. Нажмите "Попробовать еще раз" для нового запроса

## API Endpoints

- `GET /` - главная страница
- `GET /api/methods` - получить список доступных методов
- `POST /api/process-links` - обработать строку выбранными методами

## Добавление новых методов

Чтобы добавить новый метод обработки, просто добавьте его в список `METHODS` в файле `src/main.py`:

```python
METHODS = [
    Method("reverse", lambda s: s[::-1]),
    Method("uppercase", lambda s: s.upper()),
    # ... существующие методы
    Method("новый_метод", lambda s: ваша_функция(s)),  # Добавьте сюда
]
```

Метод автоматически появится на странице при следующей загрузке.
